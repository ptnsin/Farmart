# AgriHarvest Login

## วิธีรัน (Windows / Mac / Linux)

1. แตกไฟล์ zip นี้ออกมา
2. เปิด Terminal / PowerShell แล้ว cd เข้าไปในโฟลเดอร์นี้
3. รันคำสั่ง:

```bash
npm install
npm run dev
```

4. เปิดลิงก์ที่ terminal แสดง (ปกติคือ http://localhost:5173)

เสร็จแล้วจะเห็นหน้า Login ของ AgriHarvest ทันที
